# Hawkes-Process Toolkit

A Python toolkit for self-exciting point processes, built around the thesis that
financial order flow, neural spike trains, and geomagnetic/seismic cascades share
one conditional-intensity core.

**Status:** decide phase (research, not yet library code). See `ROADMAP.md` for the
decision tree (N1-N4), experiment cards (E1-E4), and manifesto stubs. No `/manifesto`
has been locked in yet — everything under `experiments/` is scratch/throwaway code
that exists to resolve open branches, not the toolkit's real API.
